## flipX 방향 처리

`MovementController`의 이동 방향에 따라 `SpriteRenderer.flipX`를 전환한다.

```
direction.x > 0 → flipX = false  (오른쪽)
direction.x < 0 → flipX = true   (왼쪽)
```

`MovementController.cs`에 추가:
```csharp
[SerializeField] private SpriteRenderer _spriteRenderer;

// Move() 내부 ClampToBounds() 호출 후:
if (_direction.x != 0f)
    _spriteRenderer.flipX = _direction.x < 0f;
```

```
[스프라이트 PNG]
  └─► Unity Importer (Point Filter, PPU 100, Lossless)
        └─► Sprite Editor 슬라이싱 (Grid 128×128)
              └─► Animation Clip 프레임 할당 (12fps)
                    └─► PetAnimator.controller State Node에 Motion 연결
                          └─► AnimationStateMachine.SetState() 호출 시 재생

[MovementController._direction]
  └─► SpriteRenderer.flipX 실시간 갱신
```
