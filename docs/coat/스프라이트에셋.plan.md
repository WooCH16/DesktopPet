# PLAN — 스프라이트에셋

> COAT 단계: PLAN
> 작성일: 2026-04-05
> 기능명: 스프라이트에셋

---

## 핵심 목표

데스크탑 펫의 5개 상태(Idle / Walk / Run / Sleep / Sit) + 클릭 반응(React)에 대응하는
스프라이트 시트와 Animation Clip을 제작하여 PetAnimator.controller에 연결한다.

---

## 요구사항

### 스프라이트 규격

- [ ] S-01 캔버스 크기: 128×128px (1 스프라이트 프레임 기준)
- [ ] S-02 배경: Magenta (#FF00FF) 단색 — colorkey 투명 처리
- [ ] S-03 픽셀 아트 스타일 권장 (선명한 외곽선, 제한된 팔레트)
- [ ] S-04 Sprite Mode: Multiple (스프라이트 시트) 또는 Single (프레임별 파일)

### 애니메이션 클립별 요구사항

| 클립 | 최소 프레임 | Loop | 설명 |
|------|------------|------|------|
| Pet_Idle  | 4 프레임 | ON  | 숨쉬기·눈 깜빡임 등 미세한 움직임 |
| Pet_Walk  | 6 프레임 | ON  | 좌우 이동 걷기 사이클 |
| Pet_Run   | 6 프레임 | ON  | Walk보다 빠른 달리기 사이클 |
| Pet_Sleep | 4 프레임 | ON  | 눈 감고 규칙적 숨쉬기 |
| Pet_Sit   | 2 프레임 | ON  | 앉아있기 (미세 움직임) |
| Pet_React | 4 프레임 | OFF | 클릭 반응 (놀람·기쁨 등) |

### Unity 임포트 설정

- [ ] I-01 Texture Type: Sprite (2D and UI)
- [ ] I-02 Pixels Per Unit: 100 (128px → 1.28 Unity unit)
- [ ] I-03 Filter Mode: Point (픽셀 아트 선명도 유지)
- [ ] I-04 Compression: None 또는 Lossless
- [ ] I-05 Sprite Sheet 사용 시 Sprite Editor로 슬라이싱

### Animation Clip 연결

- [ ] A-01 각 클립을 PetAnimator.controller 상태 노드에 Motion으로 연결
- [ ] A-02 Sample Rate: 12 fps
- [ ] A-03 Walk/Run 좌우 반전: SpriteRenderer.flipX로 방향 처리
