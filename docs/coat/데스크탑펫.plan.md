# PLAN — 데스크탑 펫

> COAT 단계: PLAN
> 작성일: 2026-04-05
> 기능명: 데스크탑 펫

---

## 핵심 목표

Windows 바탕화면 위에서 항상 최상위로 떠다니며, 사용자와 상호작용하는 투명 배경의 애니메이션 펫 애플리케이션을 Unity로 구현한다.

---

## 요구사항

### 기능 요구사항

- [ ] F-01 투명 배경 윈도우: Magenta colorkey 기반 투명도 처리, 항상 최상위(Topmost)
- [ ] F-02 애니메이션 재생: Idle / Walk / Run / Sleep / Sit 상태별 스프라이트 애니메이션
- [ ] F-03 자율 이동: 바탕화면 경계 내에서 자동으로 이동 (Walk/Run 상태)
- [ ] F-04 마우스 드래그: 사용자가 펫을 클릭하여 드래그로 이동 가능
- [ ] F-05 클릭 반응: 클릭 시 감정 표현 애니메이션 재생
- [ ] F-06 바탕화면 경계 감지: 화면 밖으로 나가지 않도록 경계 충돌 처리
- [ ] F-07 트레이 아이콘: 시스템 트레이에 아이콘 표시, 우클릭 메뉴 (종료 / 설정)
- [ ] F-08 시작프로그램 등록: Windows 시작 시 자동 실행 옵션

### 기술 요구사항

- [ ] T-01 Unity 6.3 LTS 사용
- [ ] T-02 WinAPI P/Invoke: SetWindowLong, SetLayeredWindowAttributes 등 직접 호출
- [ ] T-03 빌드 타겟: Windows x64 독립 실행형 (.exe)
- [ ] T-04 메모리 사용량: 상시 실행 기준 100MB 미만
- [ ] T-05 CPU 사용량: Idle 상태 기준 1% 미만

### UX 요구사항

- [ ] U-01 부드러운 애니메이션: 프레임 드롭 없이 60fps 재생
- [ ] U-02 자연스러운 이동: 이동 속도·방향 전환이 어색하지 않음
- [ ] U-03 펫 크기: 바탕화면에서 존재감 있되 방해되지 않는 크기 (128×128px 기준)
- [ ] U-04 우클릭 메뉴 반응: 트레이 메뉴가 500ms 이내 반응
